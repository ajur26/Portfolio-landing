ENV-MONITOR

ENV-MONITOR to webowa aplikacja IoT służąca do monitorowania warunków środowiskowych w pomieszczeniach. System zbiera dane z mikrokontrolera ESP32 oraz podłączonych czujników i prezentuje je w panelu użytkownika.


Aplikacja umożliwia monitorowanie temperatury, wilgotności, ciśnienia oraz poziomu tlenku węgla. Dane są przesyłane z urządzenia pomiarowego do backendu, zapisywane w bazie danych i prezentowane w postaci aktualnych wartości, wykresów oraz historii pomiarów.

Najważniejsze funkcje

  • Odczyt danych z czujników środowiskowych
  • Przesyłanie pomiarów z ESP32 do aplikacji
  • Prezentacja aktualnych wartości
  • Wykresy (1h / 24h)
  • Historia pomiarów
  • System alarmów dla niebezpiecznych wartości
  • Logowanie użytkownika
  • Zabezpieczenie API

Technologie

Frontend: React, JavaScript, HTML, CSS, Recharts
Backend: Python, Django, Django REST Framework
Baza danych: SQLite / PostgreSQL
IoT: ESP32, MicroPython
Czujniki: BME280, MQ-7
Inne: REST API, JWT, Git, GitHub, VPS, Nginx

Moja rola

Zaprojektowałem i zrealizowałem cały system: od części sprzętowej, przez backend i bazę danych, aż po frontend aplikacji. Stworzyłem API do obsługi pomiarów, zaimplementowałem logikę alarmów oraz komunikację między ESP32 a aplikacją webową.
